# Punic War Web Game

This project was developed at Cy École de Design in Saint-Germain-en-Laye, France, for the Web Programming course. With Joschka Mayer, Alec Mignot, Alann Samson, Arthur Chauvin.

This is a web‐based game about the Punic Wars. Coded in pure JS, HMTL, CSS.

You can find a video demo here : https://youtu.be/tLFzAnlPpHs

## How to play

Open `index.html` in your favorite browser and enjoy!
